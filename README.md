# Debarjun Maiti - Interactive Recruiter Portfolio

A premium, interactive developer portfolio designed to stand out to technical recruiters.

## Features
- **Dynamic Recruiter Pitch**: Recruiters can select their specific industry (SaaS, AI/ML, Enterprise) to instantly customize the portfolio copy and highlight relevant achievements.
- **Recruiter Assistant Q&A Chatbot**: A simulated chat experience answering recruitment-specific questions instantly.
- **3D Glowing Skill Matrix**: Interactive grid display of frontend, backend, database, and devops skills.
- **Mouse-Interactive Particle Canvas**: Fluid, lightweight canvas animation reacting to cursor hover.
- **Interactive Terminal Contact Console**: Interactive terminal interface to send emails and copy socials.

## How to Run Locally
1. Simply double-click `index.html` in your web browser, OR
2. Serve using any static server (e.g. `npx browser-sync start --server` or `python -m http.server 8000`).
